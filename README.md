# Ethan & Sophia - Wedding Invitation 💍

A beautiful, interactive digital wedding invitation built with HTML, CSS & JavaScript.

## Features
- Elegant envelope opening animation
- Romantic couple presentation
- Fun maze game (Help the groom find his bride)
- Live countdown timer
- Photo gallery
- Timeline of their journey
- Responsive design (Mobile + Desktop)

## How to Use
Just open `index.html` in any browser.
